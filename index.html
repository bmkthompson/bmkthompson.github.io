Id Therapy Project

The goal of this project was to create an AI assistant that would recommend natural remedies for common physical and mental symptoms, looking back to a time where health was instinctual and not so complicated. The aim is to reduce reliance on pharmaceuticals and suggest a more natural approach in order to avoid unwanted side effects. I created a database to store the data on the natural remedies and integrated this database into an OpenAI Assistant. I then created a user-friendly front end for the assistant.

  • AI Assistant/OpenAI API
  • Programatic CSV Manipulation
  • Site Optimization/Management

    • Stock Price Model

    Here is one of my favorite projects, being that I enjoy trading currencies in my free time. I learned how to create an LSTM Neural Network model that uses stock data from Yahoo Finance. This is the barebones model and requires personal tweaking to make consistently profitable. However, as  you can see from the graphs, the model is quite accurate right out of the box. I also created an interactive S&P500 stock map to display individual stock performances based on earnings and market cap.

  • Python
  • Data Retrieval (yfinance)
  • Pandas/Numpy Data Manipulation
  • Neural Networks
